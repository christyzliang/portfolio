Book Jackets

Description of the Project

Radio Lab is a podcasts that investigates the strange world that we live in. Three podcasts were chosen to be interpreted into book jackets. The final trilogy should represent the theme and mood of the podcasts.

Client: Student Project
Service: Print
Tools: Photoshop, Indesign

PODCASTS

THREE RENDITION

The first podcast I listened to was "Famous Tumors". Based off on that first podcast I created three different interpretations using three different mediums. Digital, string, and paint.

CHOSEN ONE

After listening to all three podcasts, I choose to create the trilogy with string. String became the one that links them together.
