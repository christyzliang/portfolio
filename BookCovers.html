Book Jackets

Description of the Project

Radio Lab is a podcast that investigates a strange world in which we live in. Three podcasts were chosen to be interpreted into book jackets. The final trilogy should represent the theme and mood of the podcasts.

Design decisions included: typography, grid systems, and design process

Client: Student Project
Service: Print
Tools: Photoshop, and Indesign

PODCASTS

THREE RENDITION

The first podcast I listened to was "Famous Tumors". Based off on that first podcast I created three different interpretations using three different mediums. Digital, string, and paint.

CHOSEN ONE

After listening to all three podcasts, I choose to create the trilogy with string. String became the one that links them together.
